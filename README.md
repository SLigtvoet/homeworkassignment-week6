# Codaisseur homework-assignment week 6

## This repository contains an API for the Tic-Tac-Toe game.

### Installation:

```
git clone https://github.com/SLigtvoet/homeworkassignment-week6
yarn install
yarn tsc -w
docker run postgres
yarn nodemon
```

### API endpoints:
```
GET /games
POST /games
PATCH /games/:id
```
